Foundational math phrase definitions

Declaring new symbols

We need a way to declare new symbols.  For example, when defining propositional logic, one must reserve symbols for "and," "or," etc.  Or when defining the real numbers, one needs to reserve symbols for R, +, etc.  This math phrase will make such declarations more readable.  To find the exact symbol named used in the internal syntax tree for a formula created using the math editor, edit the formula and use the "View meaning structure" button.

Reserve SYM as a new symbol
Define a new math phrase with the following properties.
Name: Reserve a symbol
Parameters: SYM
External representation (in HTML): Reserve SYM as a new symbol
Internal representation notation: putdown
Internal representation: :[SYM] +{"_type_Declaration":true}

Making assumptions

The following math phrase definition makes it so that you can turn any expression into an assumption in a way that presents readably in English.

Assume STATEMENT
Define a new math phrase with the following properties.
Name: Assumption
Parameters: STATEMENT
External representation (in HTML): Assume STATEMENT
Internal representation notation: putdown
Internal representation: :STATEMENT

Declaring arbitrary variables

The following math phrase definition makes it so that you can declare arbitrary variables in a way that is readable in English, and the appropriate internal flags will be set for you to communicate that meaning to the system.

Let VAR be arbitrary
Define a new math phrase with the following properties.
Name: Arbitrary variable declaration
Parameters: VAR
External representation (in HTML): Let VAR be arbitrary
Internal representation notation: putdown
Internal representation: :[VAR]

Declaring variables with specific properties

When you know a mathematical object exists with a certain property, you often want to give it a name, so that you can reason about it.  This math phrase definition makes it possible to take such a step in a document, with the appropriate internal meaning coupled to a readable English representation.

Let VAR be such that CONDITION
Define a new math phrase with the following properties.
Name: Name an object that exists
Parameters: VAR, CONDITION
External representation (in HTML): Let VAR be such that CONDITION
Internal representation notation: putdown
Internal representation: [VAR , CONDITION]

Natural deduction rules

Conjunction introduction and elimination

Assume A.  Assume B.  Conclude A∧B.

Assume A∧B.

You may conclude both A and B, as separate statements or individually.

Disjunction introduction and elimination

Assume A.

You may conclude both A∨B and B∨A, as separate statements or individually.

Assume A∨B.

Case 1:  Assume A.  Prove C.

Case 2:  Assume B.  Prove C.

Conclude C, because it's true in all cases.

Implication introduction and elimination

Assume A.  Prove B.

Conclude A⟹B.

Assume A⟹B.  Assume A.  Conclude B.

Negation introduction and elimination

Assume ¬A.  Prove both B and ¬B.

Conclude A.

Assume A.  Prove both B and ¬B.

Conclude ¬A.
