Class: SettingMetadata

SettingMetadata(name, label, defaultValue)

Every setting in the app must come with some metadata, including its name (a unique key used for saving it), its label (what prompt will be shown on screen to the user when editing the setting), and its default value (which will be its value before the user has ever edited or even seen that setting). This abstract base class supports just thos three features, but does not do anything useful with them yet, except provide generic functionality factored out of most subclasses.

This is not to be confused with SettingsMetadata (note the plural), which is for an entire collection of settings. This one (the singular) is the metadata for just one setting.
